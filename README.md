# Blog

Personal blog built with Go, React, and PostgreSQL – deployed on a self-managed Arch Linux V-Server.

## Status

🚧 Work in progress

## Tech Stack

**Backend:** Go, Gin, GORM, PostgreSQL  
**Frontend:** React, Vite, TypeScript  
**API:** OpenAPI 3.0 (API-First)  
**Infra:** Docker, Docker Compose, Nginx, Let's Encrypt, GitHub Actions
